# landing

Landing web de Reparteix.

## Missatge actual

La landing ha d'explicar Reparteix com una alternativa local-first a Splitwise amb aquests eixos:

- sense comptes ni registre
- offline i instal·lable
- dades sota control de l'usuari
- sincronització P2P xifrada entre dispositius

## Fitxers principals

- `index.html` — landing estàtica principal
- `og-image.png` — imatge social
- `favicon.svg` — favicon
